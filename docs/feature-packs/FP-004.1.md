# Feature Pack 004.1 - Knowledge Graph Foundation

## Objective

Add a pure Knowledge Graph layer on top of parsed Markdown intelligence.

## Architecture

```text
Filesystem
-> Markdown Parser
-> KnowledgeDocument[]
-> Knowledge Graph Service
-> Nodes + Edges + Backlinks
-> Search / Dashboard / Products / AI later
```

## Delivered

- Knowledge graph models:
  - `KnowledgeGraphNode`
  - `KnowledgeGraphEdge`
  - `KnowledgeGraph`
- Link resolver for wiki links and embeds.
- Graph builder from parsed `KnowledgeDocument[]`.
- Backlink lookup.
- Outgoing link lookup.
- Related-note ranking.
- In-memory graph cache with `refreshGraph()`.
- Graph API routes:
  - `GET /api/graph`
  - `GET /api/graph/related?q=<path>`
  - `GET /api/graph/backlinks?q=<path>`
- Search scoring can optionally receive graph context.
- Tests for broken links, circular links, backlinks, outgoing links, related notes, duplicate titles, empty vault, and large vault.

## Notes

The graph layer is independent of the filesystem and does not duplicate parser logic. It consumes parsed `KnowledgeDocument` objects from the existing vault indexer.
