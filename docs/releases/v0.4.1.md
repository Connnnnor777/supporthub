# v0.4.1 - Knowledge Graph Foundation

## Added

- Knowledge Graph service over parsed `KnowledgeDocument` records.
- Wiki-link and embed edges.
- Backlinks, outgoing links, and related-note ranking.
- Cached graph access with future-ready `refreshGraph()`.
- Graph APIs for nodes, edges, related notes, and backlinks.
- Graph unit tests.

## Changed

- Began migration from `VaultNote` toward `KnowledgeDocument` with backward-compatible type aliases.
- Search scoring can optionally use graph context for additive boosts.

## Compatibility

- Dashboard remains unchanged.
- Search API remains stable.
- Vault API remains stable.
