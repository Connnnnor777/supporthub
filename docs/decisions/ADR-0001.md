# ADR-0001

## Title

Obsidian Vault as the Single Source of Truth

## Status

Accepted

## Date

2026-07-14

## Decision

SupportHub will use the Obsidian vault as its primary knowledge source.

The application will never duplicate or permanently store documentation outside the vault.

## Rationale

- One location for knowledge
- Easy backup
- Obsidian remains fully usable
- SupportHub becomes the interface, not the database

## Consequences

- Services read from the vault.
- APIs expose service data.
- UI never reads files directly.
